# Coffee Shop Sales Analysis
## Overview
Analysis of coffee shop sales data using Microsoft Excel and Power Query.
## Tools
- Microsoft Excel
- Power Query
- Pivot Tables
## Results
- Cleaned and prepared the dataset
- Analyzed sales trends and revenue
- Identified peak sales hours and top-performing product categories
- Developed recommendations to improve business performance
